# alzheimer-parkinson-ai
Sistema de detecção precoce de Alzheimer e Parkinson utilizando Inteligência Artificial aplicada à análise de fala e escrita. O projeto utiliza dados públicos (VOICED, ADReSSo) e aplica técnicas de NLP e processamento de sinais de áudio para extrair padrões linguísticos e acústicos característicos de pacientes com diagnóstico clínico.
